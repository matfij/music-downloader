### Music Downloader
Simple and configurable CLI app for downloading YouTube videos.

## Requirements
 - Python 3.6.x-3.9.x

## Setup (Windows)
 - cdm: `py -m venv .venv`
 - cmd: `.venv\Scripts\activate`
 - cmd: `pip install -r requirements.txt`
 - adjust config.json file to your needs
 - cmd: `py run.py`
